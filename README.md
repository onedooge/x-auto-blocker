# X Auto Blocker 🛡️

自动检测并屏蔽X（Twitter）上发布不当内容的账号。

## 功能

- ✅ 自动扫描时间线上的推文
- ✅ 关键词检测（支持中英文）
- ✅ 自动屏蔽账号 或 仅隐藏推文（二选一）
- ✅ 自定义关键词（增删改）
- ✅ 统计已屏蔽数量
- ✅ 开关随时切换

---

## 安装方法（Chrome / Edge）

1. 下载并解压本插件文件夹
2. 打开浏览器，地址栏输入：
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
3. 右上角开启「**开发者模式**」
4. 点击「**加载已解压的扩展程序**」
5. 选择本文件夹（x-auto-blocker）
6. 安装完成 ✓

---

## 使用方法

1. 打开 x.com，登录账号
2. 点击浏览器右上角的插件图标
3. 确认状态显示「运行中」
4. 插件会自动扫描并处理不当推文

### 模式说明

| 模式 | 说明 |
|------|------|
| 自动屏蔽账号（默认）| 检测到后自动点击"屏蔽"，永久屏蔽该账号 |
| 仅隐藏推文 | 只在本地隐藏，不进行屏蔽操作（更安全） |

---

## 注意事项

- 自动屏蔽功能依赖X页面的DOM结构，如X更新页面可能需要调整选择器
- 建议先使用「仅隐藏推文」模式测试效果，确认无误后再开启自动屏蔽
- 关键词可在弹窗中随时增删
